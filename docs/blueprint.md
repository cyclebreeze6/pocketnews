# **App Name**: Pocketnews TV

## Core Features:

- Channel Creation: Admin can create TV channels with names and descriptions.
- Video Upload via YouTube API: Admin uploads videos by submitting YouTube links; the app uses the YouTube API to fetch video data like title, description, and thumbnails.
- Video Categorization: Admin categorizes videos into different channels.
- User Video Display: Users can browse channels and watch videos on a clean, simple UI similar to Haystack.tv.
- Admin Dashboard: Admin has access to a dashboard for managing videos, channels, users, and other admin tasks.
- Analytics Dashboard: Admin analytics dashboard shows key metrics such as video views, user engagement, and channel performance. It uses AI as a tool to interpret what affects the engagement numbers.
- User and Admin Management: Admin can manage users (add/remove/suspend) and assign admin roles to other users.

## Style Guidelines:

- Primary color: Deep blue (#1A237E) for a trustworthy, news-oriented feel.
- Background color: Very dark desaturated blue (#0A0F2B) for a comfortable contrast in a dark color scheme.
- Accent color: A saturated cyan (#00BCD4) for highlights, links, and interactive elements.
- Body and headline font: 'Inter' (sans-serif) for a clean and modern reading experience.
- Use simple, modern icons for navigation and video controls.
- Implement a clean and grid-based layout, similar to Haystack.tv, with a focus on video thumbnails and clear navigation.
- Subtle transitions for loading content and navigating between channels.
